# README
## ImageMergeTool
一个简洁的图片拼接工具，支持横向拼接、纵向拼接和 2×2 网格拼接。

---

### 功能特性

- **横向拼接** — 将多张图片按水平方向拼接
- **纵向拼接** — 将多张图片按垂直方向拼接
- **2×2 网格拼接** — 将 4 张图片拼接为 2×2 方阵
- 拖拽导入图片，操作简单直观
- 本地运行，无需上传，保护隐私

---

### 下载安装

#### Windows 用户
直接前往 [Releases](https://github.com/turnaoo/ImageMergeTool/releases) 页面下载 `.exe` 文件，双击即可运行。

> ⚠️ 目前仅提供 Windows 版本。如需 macOS 或 Linux 版本，请提交 [Issue](https://github.com/turnaoo/ImageMergeTool/issues)，或参考下方自行编译。
